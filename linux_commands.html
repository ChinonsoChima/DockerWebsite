Linux Commands

cd := change directory // Navigate between directories.

ls := list directory contents // List files within a directory, add options such as '-a' for a more detailed list.

mkdir := make directory // 'Create a New Folder'... *nix style!

pwd := present working directory // View the current working directory.

cat := view the contents of a file // Stands for 'concatenate' -- Shows the contents of a file directly on the Terminal.

touch := create an empty file or update the access date of a file. // Literally touches a file by updating the accessed time if the file exists or creating a file if the specified parameters[filename] does not exist.

chmod := change/modify permissions of a file // Allow or disallow user access to files based on their permissions.

rm := remove // Remove a file or folder.

cp := copy // Copy a file

mv := move a file // Can also be used to rename a file.

find := find a file // Find a file, can also pipe with grep for enhanced results.

grep := search for strings // Match string patterns in searches-- can be combined with Regular Expressions.

vi := open a text editor, or open a file in a text editor.
